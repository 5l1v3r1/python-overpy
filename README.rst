Python Overpass Wrapper
=======================

A Python Wrapper to access the Overpass API.

Have a look at the `documentation`_ to find additional information.

Install
-------

**Requirements:**

* Python 2.7 or Python >= 3.2


**Install:**

.. code-block:: console

    $ pip install overpy
